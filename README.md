# 📸 Responsive Image Slider

A modern and responsive **Image Slider** built using **HTML, CSS, JavaScript, and Bootstrap 5**. This project demonstrates how to create a smooth, mobile-friendly image carousel with navigation controls and responsive design.

## 🚀 Live Demo

🔗 **Website:** https://kamrul2211.github.io/image-slider/

## 📌 Features

* 🎞️ Responsive Image Slider
* 📱 Mobile-Friendly Design
* ⬅️ Previous & Next Navigation
* 🎯 Smooth Slide Transitions
* ⚡ Fast and Lightweight
